Python Primitive Types

Posted on 2019-03-20 | In Python | Comments:

Integer

Integers in Python3 are unbounded. The maximum integer representable is a function of the available memory.
There is no concept of an unsigned shift/integer in Python since integers have infinite precision. (Need to be careful about shifting here).

float

Floats, unlike integers, are not infinite precision. float('inf') and float('-inf') are used to refer to infinity. These values are comparable to integers and can be used to create pseudo max-int and pseudo min-int.

When comparing floating point values use math.isclose(). It is robust as for when comparing very large values, and can handle both absolute and relative differences.
